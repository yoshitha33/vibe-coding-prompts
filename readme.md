## Common Copilot Gotchas & Solutions

| Issue | Solution |
|-------|----------|
| Copilot generates wrong code | Refine your prompt with more context |
| Response seems incomplete | Accept and continue typing your needs |
| Refuses to generate code | Rephrase without suspicious keywords |
